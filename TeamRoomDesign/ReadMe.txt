
First create an application using the Teamroom template, 
then use the Designer team development plugin 
to create an association with this project,
and to import from this project into your NSF.
You can use allDocuments.xsp to CRUD documents.
This application depends on extlib, so that must be installed to use it.
You may also have to change the Access Control settings,
and in the Access Control dialog switch to the Advanced tab
and choose "Maximum internet name and password": "Manager".
